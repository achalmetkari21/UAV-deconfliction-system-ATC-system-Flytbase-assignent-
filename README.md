# Drone Traffic Management System (ATC Simulation)

## Overview

This project presents a simulation of a drone traffic management system inspired by Air Traffic Control (ATC). It is designed to manage multiple drones operating within a shared airspace by detecting potential conflicts and generating safe trajectories.

---

## Features

* Multi-drone simulation with support for 30 or more drones
* Real-time conflict detection between drone paths
* Trajectory planning and deconfliction strategies
* Interactive dashboard for visualization (built using Streamlit)
* Scenario-based simulation for testing different conditions

---

## Project Structure

infrared-station/ → Dashboard and visualization components
silent-stellar/ → Core logic, services, and trajectory management

---

## Technology Stack

* Python
* Streamlit
* NumPy
* Matplotlib

---

## Running the Project

Install dependencies:
pip install -r requirements.txt

Run the application:
streamlit run infrared-station/atc_dashboard/dashboard.py

---

## Applications

* Drone traffic management systems
* Smart city airspace coordination
* Robotics and autonomous systems simulation
